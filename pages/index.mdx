## What can I do?

**Morgen API** allows you to integrate any of our supported calendars into your software. Learn more here: [Morgen API Reference](/introduction)

_Notice that the Morgen APIs are in early stage and only the APIs to operate on calendar data is available at the moment._

## Get Your API Key

To use the Morgen API, you need to get an API key. You can get one by signing up at [https://platform.morgen.so](https://platform.morgen.so).

Learn more in the [Authentication](/authentication) section.
