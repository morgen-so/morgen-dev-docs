# Morgen API Authentication

## Get your API key
<Steps>
    ### Sign in to Morgen Assist:

    Create a new account or sign in to [Morgen Assist](https://platform.morgen.so).
    If you already have a Morgen account, you can use it to sign in to Morgen Assist.

    ### Get your API key:
    You can get your API key navigating to [Developers API](https://platform.morgen.so/developers-api) page.

    ### Copy your API key:
    In the page you will be able to copy your API key, keep it safe.<br/>
    If you believe your API key has been leaked, you can regenerate it.

</Steps>
