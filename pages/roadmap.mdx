# Roadmap

This is a tentative roadmap for the next few months. We will update this page as we make progress.

| Feature                                                                    | Supported Providers         | Availability |
| :------------------------------------------------------------------------- | :-------------------------- | :----------- |
| Authentication                                                             | Google, Office 365          | ✅           |
| List, create, update, delete, and RSVP simple events                       | Google, Office 365          | ✅           |
| List recurring events                                                      | Google, Office 365          | ✅           |
| Update, delete, and RSVP to a single instance of recurring events          | Google, Office 365          | ✅           |
| Background regular credentials check                                       | All                         | ✅           |
| Basic admin dashbord                                                       | All                         | ✅           |
| Creation of Google Meet links on Google Calendar events                    | Google                      | ✅           |
| Creation of Teams meeting links on Office 365 Calendar events              | Office 365                  | ✅           |
| Webhooks for event changes                                                 | Google, Office 365          | Soon         |
| Create recurring events                                                    | Google, Office 365          | Soon         |
| Creation of Zoom meeting links on all calendars                            | Zoom                        | Soon         |
| Update, delete, and RSVP to all and future occurrences of recurring events | Google, Office 365          | Soon         |
| Support for CalDAV calendars                                               | iCloud and others           | Soon         |
| Support for Exchange on Premise calendars (EWS API)                        | Exchange                    | Soon         |
| Creation of Webex meeting links on all calendars                           | Webex                       | Soon         |
| Tasks (Todoist, Google Tasks, Microsoft To Do, etc.)                       | Todoist, Google Tasks, etc. | Soon         |
